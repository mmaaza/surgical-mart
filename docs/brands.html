¬© Brands API

Manage product brands with featured status and comprehensive brand information

Overview

The Brands API provides comprehensive brand management for organizing products by manufacturer or brand identity. Brands can be featured for homepage display and include rich metadata for SEO optimization.

üîë Key Features

  • Featured Brands: Mark brands as featured for homepage and promotional display
  • SEO Optimization: Meta titles, descriptions, keywords, and custom slugs
  • Status Management: Active/inactive status control with toggle functionality
  • Search & Filter: Search by name, description, and tags with featured filtering
  • Rich Media: Brand logos and images with optimized display
  • Tag System: Organize brands with custom tags for easy categorization

üåê Public API Endpoints

These endpoints are publicly accessible and don't require authentication. Perfect for frontend applications displaying brands to customers.

GET /brands Public

Retrieve all active brands with optional filtering

Query Parameters:

Parameter Type Required Description
featured boolean No Filter by featured status (true/false)
search string No Search in brand name, description, and tags

Code Examples

JavaScript - Fetch Brands

// Get all active brands
const getBrands = async (filters = {}) => {
  const params = new URLSearchParams(filters);
  const baseUrl = process.env.NODE_ENV === 'production' 
    ? 'https://dentalkartnepal.com/api' 
    : 'http://localhost:5000/api';
    
  const response = await fetch(`${baseUrl}/brands?${params}`);
  return response.json();
};

// Get featured brands only
const getFeaturedBrands = async () => {
  return getBrands({ featured: 'true' });
};

// Search brands
const searchBrands = async (query) => {
  return getBrands({ search: query });
};

// Usage
const allBrands = await getBrands();
const featuredBrands = await getFeaturedBrands();
const searchResults = await searchBrands('medical');

console.log('All brands:', allBrands.data);
console.log('Featured brands:', featuredBrands.data);
console.log('Search results:', searchResults.data);

JavaScript - Create Brand

const createBrand = async (brandData) => {
  const token = localStorage.getItem('authToken');
  const baseUrl = process.env.NODE_ENV === 'production' 
    ? 'https://dentalkartnepal.com/api' 
    : 'http://localhost:5000/api';
  
  const response = await fetch(`${baseUrl}/brands`, {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${token}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify(brandData)
  });
  
  return response.json();
};

// Usage
const newBrand = await createBrand({
  name: 'HealthTech Innovations',
  description: 'Cutting-edge medical technology solutions',
  picture: '/uploads/brands/healthtech-logo.jpg',
  tags: ['medical', 'technology', 'innovation'],
  featured: true,
  metaTitle: 'HealthTech Innovations - Medical Technology Solutions',
  metaDescription: 'Discover innovative medical technology solutions from HealthTech Innovations',
  keywords: 'medical technology, healthcare innovation, medical devices'
});

React Hook - Brands

import { useState, useEffect } from 'react';

const useBrands = (filters = {}) => {
  const [brands, setBrands] = useState([]);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    const fetchBrands = async () => {
      try {
        setLoading(true);
        const params = new URLSearchParams(filters);
        const baseUrl = process.env.NODE_ENV === 'production' 
          ? 'https://dentalkartnepal.com/api' 
          : 'http://localhost:5000/api';
          
        const response = await fetch(`${baseUrl}/brands?${params}`);
        const data = await response.json();
        
        if (data.success) {
          setBrands(data.data);
        } else {
          setError(data.error);
        }
      } catch (err) {
        setError(err.message);
      } finally {
        setLoading(false);
      }
    };

    fetchBrands();
  }, [JSON.stringify(filters)]);

  return { brands, loading, error };
};
